# Give a Kudo

Help the user recognize an associate with a kudo.

## Steps

1. **Who deserves recognition?** Ask for the associate's name. Use `list_associates` to find them and confirm.

2. **What did they do?** Ask the user to describe what the associate did well. Use their response to fill the kudo details.

3. **Review and confirm** — Show a preview before creating.

4. **Create** — Call `create_associate_kudo` with the collected data. Show the result.

Keep it positive and celebratory. Recognition matters.
